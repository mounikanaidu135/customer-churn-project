# Customer Churn Prediction

This project predicts whether a customer will leave a service using Logistic Regression.

## Technologies Used

- Python
- Flask API
- Streamlit
- MongoDB
- Scikit-learn

## Project Structure

backend/ → Flask API  
frontend/ → Streamlit UI  
model/ → ML training and saved model  
dataset/ → Kaggle dataset

## How to Run

1. Train model
python model/train_model.py

2. Start backend
python backend/app.py

3. Run frontend
streamlit run frontend/app.py
